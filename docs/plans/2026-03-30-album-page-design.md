# 相册详情页设计文档

> 日期：2026-03-30
> 状态：已确认

---

## 1. 概述

对相册详情页进行视觉升级，延续首页的电影胶片风格，融合报纸/杂志风排版，让婚礼照片有"时光印记"的质感。

**设计原则：**
- 响应式优先（手机 + PC 双端）
- 照片尺寸不一，有的长有的高，所有布局必须原生适配原比例
- 与首页视觉语言一致（胶片感、暖色调、角落框架）

---

## 2. 整体视觉语言

| 元素 | 样式 |
|------|------|
| 背景色 | `#faf9f7` + grain texture（同首页） |
| 角落框架 | 左上/右上 L 形深色线条（同首页） |
| 主色调 | 金色 `#b48c50`，深褐 `#1a1a1a`，暖白 `#faf9f7` |
| 字体 | Playfair Display（标题）+ Lora（正文） |

---

## 3. 顶部导航栏

固定顶部，深色背景 `#1a1a1a`，居中布局：

- **左侧**：`← 返回`，链接回首页，金色 `#b48c50`
- **中间**：当前相册标题，Playfair Display，白色
- **右侧**：布局类型标签（masonry / grid / timeline），金色小写字母

---

## 4. 相册信息区

位置在顶部导航栏下方，内容居中：

```
              ✦ 连云港 ✦
           2024年的旅行记录

     ─────────── ◆ ───────────
```

**样式：**
- 标题：Playfair Display，~2.5rem，金色 `#b48c50`，居中
- 描述：Lora 斜体，~0.9rem，淡灰色 `#888`，居中
- 分隔线：1px 金色渐变，两侧透明，中间 ◆ 菱形装饰

**动效：**
- 页面加载时标题淡入上浮（`opacity 0→1，translateY 20px→0`，800ms）
- 装饰线延迟 300ms 从中间向两侧展开（`scaleX 0→1`）

**响应式：**
- 手机端上下间距收窄，标题 `~2rem`

---

## 5. Masonry 瀑布流布局（胶片齿孔版）

### 设计

每张照片左侧有一条 **5px 宽的深色条**，上面均匀分布圆角矩形齿孔（模拟 35mm 胶片边缘）。

```
┌─[ ○  ○  ○ ]──────────────────┐
│                              │
│        照片内容               │
│                              │
└──────────────────────────────┘
```

- 齿孔条宽度：5px，深褐色 `#1a1a1a`
- 齿孔：4px × 6px，圆角 2px，间距 ~20px，金色 `rgba(180, 140, 80, 0.5)`
- 照片间距：PC 20px / Mobile 12px
- 列数：PC 4列 → 平板 3列 → 手机 2列

### 实现

CSS `columns` 属性实现瀑布流，无需 JavaScript。每张照片 `break-inside: avoid`。齿孔条用 `::before` 伪元素定位到 `.photo-item` 左侧。

### 照片尺寸适配

天然支持不同高宽比，保持原比例纵向排列，无需裁剪。

---

## 6. Grid 网格布局

### 设计

**PC 端（> 1024px）：** 4 列，照片按原始比例展示（不裁剪），最大宽度统一，高度自然。`border: 1px solid #e0ddd8`，圆角 4px，间距 12px。

**平板（640-1024px）：** 3 列

**手机（< 640px）：** 2 列，间距收窄到 8px

### 实现

同样使用 CSS `columns` 实现自由高度网格（与 Masonry 共用一套样式），只调整列数和间距。

### 照片尺寸适配

不强制 `aspect-ratio`，`object-fit: contain`，原始比例展示，两侧留白吸收高度差。

---

## 7. Timeline 时间线布局

### 设计

- 页面中央一条细细的金色竖线 `rgba(180, 140, 80, 0.25)`
- 每张照片交替出现在线条左侧和右侧（PC 端）
- 每张照片旁有日期标注（Lora 斜体，金色 `#b48c50`）
- Mobile 端：统一靠左，线条移至左侧固定位置

### 照片高度处理

每行照片最大高度统一（如 `max-height: 400px`），超出部分隐藏或缩放，保持中线笔直。左右交替通过 `flex-direction: row-reverse` 实现。

---

## 8. 灯箱（Lightbox）

### 触发

点击任意照片卡片打开全屏灯箱。

### 结构

```
┌─────────────────────────────────────────────────────┐
│ [03 / 87]                              [×]         │  ← 顶部栏
│                                                     │
│                                                     │
│                    [  照片  ]                       │  ← 居中照片
│                                                     │
│                                                     │
│  [←]                    [缩略图条]            [→]  │  ← 底部导航
└─────────────────────────────────────────────────────┘
```

### 样式

| 元素 | 样式 |
|------|------|
| 背景 | 全屏半透明黑 `rgba(0, 0, 0, 0.92)` |
| 照片 | 居中，原比例，`max-width: 90vw`，`max-height: 85vh` |
| 顶部序号 | 金色 Playfair Display，`03 / 87` |
| 关闭按钮 | 右上角 `×`，半透明白色，hover 变实白 |
| 左右箭头 | 半透明白色圆圈，hover 实心 |
| 缩略图条 | 底部一行，当前照片金色边框高亮，可左右滑动 |

### 动效

- 打开：`opacity 0→1，300ms ease-out`
- 关闭：反向淡出
- 切换：淡入淡出 `opacity 0→1→0`

### 交互

- 点击背景 / 按 ESC / 点击 × → 关闭
- 左右箭头 → 切换上一张/下一张
- 底部缩略图点击 → 跳转对应照片
- 键盘左右方向键 → 切换照片

## 9. 响应式断点

| 场景 | Masonry 列数 | Grid 列数 | Timeline |
|------|------------|---------|---------|
| 手机 < 640px | 2 列 | 2 列 | 统一靠左 |
| 平板 640-1024px | 3 列 | 3 列 | 交替 |
| PC > 1024px | 4 列 | 4 列 | 交替左右 |
